## 説明文 (日本語)

服薬スケジュールを簡単に管理できるアプリです。

自分や家族の服薬スケジュールをガントチャートで一覧表示。処方日と日数を入力するだけで、服薬終了日を自動計算します。

主な機能:
- ガントチャートで服薬期間を視覚的に管理
- 処方開始日と日数から終了日を自動計算
- 残り日数をリアルタイム表示
- メモ機能で服薬に関する情報を記録
- 7種類のカラーテーマで好みの見た目に
- 共有機能で家族間の服薬情報を簡単にやり取り
- URLスキーム対応で共有リンクからワンタップ取り込み
- データはすべて端末内に保存（サーバー送信なし）

料金プラン:
- 無料: ガントチャート3件まで
- ベーシック (¥300): 10件まで
- プレミアム (¥1,200): 無制限

## 説明文 (英語)

Easily manage medication schedules for yourself and your family.

View medication schedules at a glance with an intuitive Gantt chart. Simply enter the prescription start date and number of days to automatically calculate the end date.

Key Features:
- Visual medication schedule management with Gantt chart
- Automatic end date calculation from start date and days
- Real-time remaining days display
- Memo feature for medication notes
- 7 color themes to customize appearance
- Share medication info with family members easily
- One-tap import via URL scheme sharing
- All data stored locally on device (no server transmission)

Plans:
- Free: Up to 3 schedules
- Basic ($2.99): Up to 10 schedules
- Premium ($9.99): Unlimited schedules
